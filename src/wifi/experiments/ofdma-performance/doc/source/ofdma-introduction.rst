Overview
========

The document provides validation and initial testing results of the
ns-3 WiFi 802.11ax OFDMA feature.

This document is written in Python Sphinx markup and is maintained in
the ns-3 folder ``src/wifi/experiments/ofdma-performance/doc``.  

Outline
=======

* Section 2 provides an overview of the design of the OFDMA implementation.

* Section 3 provides instructions on how to run the ns-3 programs.

* Section 4 provides in-depth results from different studies.

* Section 5 describes open issues and future work.
